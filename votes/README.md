# Vote records

One Markdown file per included, material European Parliament vote. File name convention:

`YYYY-MM-DD--procedure-reference--short-slug.md`

Required fields:

- date and sitting;
- procedure / document reference;
- official roll-call URL;
- motion or amendment wording;
- whether raw `for` maps to `in favor` or `against` Chat Control;
- raw result totals;
- data extraction source and retrieval date.
