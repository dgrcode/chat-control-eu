# Source ledger

This directory documents primary and supporting sources used in the dataset.

Each tracked vote must have:

- a European Parliament procedure reference;
- a roll-call result URL (prefer XML or HTML annex);
- the retrieval date;
- a short statement of what the vote decided; and
- a note explaining how raw vote directions were normalized.

Every tracked sitting must also retain its official attendance-register URL and a derived matching report for MEPs with no named roll-call entry. Sitting attendance never proves chamber presence at the precise ballot or the reason for a non-vote.

Secondary sources may make exploration and extraction easier but cannot be the sole evidence for a published vote classification.

See `../METHODOLOGY.md` for source precedence and corrections.
